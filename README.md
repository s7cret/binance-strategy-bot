# Binance Strategy Bot

Telegram bot for researching, backtesting, and analyzing Binance trading strategies with self‑learning parameter tuning.

## Stack
- Python (async)
- aiogram (Telegram)
- SQLite (local storage)

## Structure
- `src/` — application code
- `tests/` — unit/integration tests
- `docs/` — documentation
- `plans/`, `notes/`, `reports/` — project artifacts
- `data/` — local data (ignored by git)

## Status
In development.
